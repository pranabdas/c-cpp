We will learn through writing code. It is customary start with "Hello world!"
program when learning a new programming language. This program simply prints a
message in the terminal indicating the code ran successfully.

Compile the code:
```bash
# in case of C
gcc filename.c

# in case of C++
g++ filename.cpp
```

It will produce an executable (binary) file named `a.out`. We can run the
executable by typing:
```bash
./a.out
```

If you will to name your program something other than `a.out`, you can specify
a custom name using `-o` flag:
```bash
gcc filename.c -o my_program
```
